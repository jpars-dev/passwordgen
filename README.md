# The Password Generator

A refined password generator with no memory. Your passwords are generated locally in your browser and never stored or sent anywhere. Something useful, thoughtfully made.

---

Adjust length from 4 to 32 characters. Toggle lowercase, uppercase, numbers, and symbols. Click the password to copy it. Your preferences are remembered locally.

## Run locally

```bash
git clone https://github.com/jpars-dev/passwordgen.git
cd passwordgen
npm install
npm run dev
```

## Built with

Next.js · React · Tailwind CSS · GSAP
