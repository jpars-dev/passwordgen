# The Password Creator

A modern, cyberpunk-themed password generator built with Next.js and enhanced with GSAP animations. This application allows users to create secure, customizable passwords with a sleek and interactive user interface.

## Features

- **Customizable Password Generation**
  - Toggle lowercase letters
  - Toggle uppercase letters
  - Toggle numbers
  - Toggle special symbols
  - Adjustable password length

- **Interactive UI Elements**
  - Copy to clipboard functionality
  - Real-time password generation
  - Visual feedback for user actions
  - Mouse-following radial gradient effect

- **Modern Design**
  - Cyberpunk theme
  - Responsive layout for all devices
  - Smooth GSAP animations
  - Clean, minimalist interface

## Technology Stack

- **Framework**: Next.js
- **Styling**: TailwindCSS with DaisyUI
- **Animations**: GSAP (GreenSock Animation Platform)
- **UI Components**: Custom components with DaisyUI

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/jpars-dev/passwordgen.git
```

2. Install dependencies:
```bash
cd passwordgen
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Usage

1. Adjust the password settings using the toggle switches:
   - Enable/disable lowercase letters
   - Enable/disable uppercase letters
   - Enable/disable numbers
   - Enable/disable special symbols

2. Use the slider to set your desired password length

3. Click "Generate Password" to create a new password

4. Click the copy icon to copy the password to your clipboard

## Development

The project structure follows Next.js conventions:
- `/src/app` - Main application pages
- `/src/components` - Reusable React components
- `/public` - Static assets
